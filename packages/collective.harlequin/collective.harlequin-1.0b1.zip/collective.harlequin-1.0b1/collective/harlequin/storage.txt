Setup some data, like some extracted by z3cform

    >>> data = {'templateId':u'configurable_browserview'}
    >>> data['size'] = u'400x200'

Let create an annotation storage and store data

    >>> from collective.harlequin import storage
    >>> hconfig = storage.AnnotationStorage(self.folder)
    >>> hconfig.get()
    {}
    >>> hconfig.set(data)

Now let's check every things ok

    >>> rconfig = hconfig.get()
    >>> len(rconfig.keys())
    2
    >>> rconfig['templateId']
    u'configurable_browserview'
    >>> rconfig['size']
    u'400x200'
