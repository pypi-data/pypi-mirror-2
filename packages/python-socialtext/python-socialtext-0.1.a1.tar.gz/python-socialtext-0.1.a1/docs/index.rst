Python bindings for the Socialtext ReST API
===========================================

A python client for the Socialtext ReST API.

.. important::
	This is an alpha release (0.1.0) of python-socialtext. The API only supports basic operations for a small number of resources. If you find a bug in the code or documentation, please file a bug on GitHub.

	See Contributing_ for more information.

You will need a user account on a Socialtext appliance.

.. seealso::
	For detailed information about the API, please see the `Socialtext ReST Documentation`__
	
	__ https://www.socialtext.net/st-rest-docs/
