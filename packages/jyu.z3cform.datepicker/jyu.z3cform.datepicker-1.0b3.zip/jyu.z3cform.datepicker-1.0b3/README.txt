jQuery Datepicker
=================

Integrates `Keith Wood's jQuery Datepicker`__ for z3c.form on Plone by

1. registering datepicker javascript and a clean custom stylesheet
2. registering viewlet (for requests implementing `IFormLayer`) to
   - render datepicker's localization for supported languages
   - render integration javascript to `collective.z3cform.datetimewidget`.

No dependency on `jquery.ui`.

Feel free to fork and complete.

__ http://keith-wood.name/datepick.html
