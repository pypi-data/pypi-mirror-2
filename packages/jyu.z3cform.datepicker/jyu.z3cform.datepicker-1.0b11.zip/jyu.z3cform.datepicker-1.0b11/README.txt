jQuery Datepicker
=================

Integrates `Keith Wood's jQuery Datepicker`__ for z3c.form on Plone by

1. registering plugin's JavaScripts and a clean custom stylesheet
2. registering viewlet to render integration JavaScript for `collective.z3cform.datetimewidget`.

No dependency on `jquery.ui`.

__ http://keith-wood.name/datepick.html
