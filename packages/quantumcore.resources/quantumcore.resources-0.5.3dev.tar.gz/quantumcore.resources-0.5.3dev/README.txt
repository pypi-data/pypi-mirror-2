Introduction
============

:mod:`quantumcore.resources` is able to manage resources such as CSS or Javascript files.
This means that it's able to merge and pack those files together and provide a caching
friendly naming scheme. 

For more information see http://quantumcore.org/docs/quantumcore.resources/
