Introduction
============

This is a plugin for `TinyMCE`__ editor for Plone.
__ http://plone.org/products/tinymce/

It is trying to add a "place gallerybutton" for collective.plonetruegallery and collective.gallery
You must whitelist "iframe' in safe-html
You must also enable "background" and "overflow" id you want to use those options
