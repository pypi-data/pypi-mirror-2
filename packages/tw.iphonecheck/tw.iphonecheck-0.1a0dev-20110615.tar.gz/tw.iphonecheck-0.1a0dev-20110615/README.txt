
tw.iphonecheck documentation
============================


tw.iphonecheck is a tosca widget wrapper around jquery iphone style checkbox plugin
 which can be found here :

`http://awardwinningfjords.com/2009/06/16/iphone-style-checkboxes.html <http://awardwinningfjords.com>`_

this widget derives from tw.forms.CheckBox and can be used in the same way
