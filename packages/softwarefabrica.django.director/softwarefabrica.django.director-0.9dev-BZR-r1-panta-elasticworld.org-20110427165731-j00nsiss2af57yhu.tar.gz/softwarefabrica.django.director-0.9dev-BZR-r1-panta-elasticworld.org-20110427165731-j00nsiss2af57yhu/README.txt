softwarefabrica.django.director library
=======================================

This library provides an advanced Django model registration
facility for SoftwareFabrica `Django`_ applications and projects.

Please refer to the `overview`_ and the `documentation`_.

Happy coding!

.. _`overview`: docs/overview.html
.. _`documentation`: docs/index.html
.. _`Django`: http://www.djangoproject.com
