Testing PloneFilesZip

Additional requirements
#######################

Running test cases resquires the following additional Products:

* PloneTestCase

* ATContentTypes 0.2 on Plone 2.0.5. Use bundled ATContentTypes with Plone 2.1 +

* AttachmentField 1.3 +

* PloneExFile 3.x +

Windows
#######

* Copy "runtests.bat.example" to "runtests.bat"

* Change "runtests.bat" as indicated in its comments

* Run it...

Unix
####

* Copy "runtests.sh.example" to "runtests.sh"

* Change "runtests.sh" as indicated in its comments

* Run it...
