====================
jFlow documentation
====================

.. rubric:: Quantitative financial library and econometric analysis

The jFlow project is aimed at providing a pluggable software framework for quantitative finance. 
It is an opensource library distributed under the very liberal BSD licence.

jFlow is written in Python and contains some modules written in C++.
It also come with a battery included django site for testing, visualising data and so forth.

Finance is an area where well-written open-source projects could make a tremendous difference.
