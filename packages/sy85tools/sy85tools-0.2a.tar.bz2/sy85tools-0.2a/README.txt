sy85 - A toolset for Yamaha SY85 files
======================================

:Author: Christopher Arndt
:Date: 2010-04-11


Overview
--------

This distribution contains a Python package, named 'sy85', for handling MIDI
sysex dumps and disk files produced and read by the Yamaha SY85 line of
synthesizers. It also comes with a few command line tools which rely on this
packages to do their work.

For more infomration  please see the help messages of the respective programs
and the module doctrings.

Please note that this software is still in early development and considered
alpha quality. Use it at your own risk!
