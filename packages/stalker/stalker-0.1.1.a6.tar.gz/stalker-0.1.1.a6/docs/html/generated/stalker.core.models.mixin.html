stalker.core.models.mixin¶

stalker.core.models.mixin¶

This module contains the Mixins (ta taaa).

Mixins are, you know, things that we love. Ok I don’t have anything to write, just use and love them.

For SQLAlchemy part of the mixins (tables and mappers) refer to the mixin. There is a corresponding type for every mixin implemented in this module. Also the documentation explains how to mixin tables and mappers.
